# Gossipsub Simulator

![Gossipsub Simulator](https://github.com/dozyio/gossipsub-simulator/blob/main/docs/images/screenshot.png?raw=true "Gossipsub Simulator")

## Usage

### Build Bootstrap/Relay and Gossipsub Nodes
```
make build
```

### Run UI
```
make ui
```

### Run Controller
```
make controller
```
